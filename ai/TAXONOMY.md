# Error/Task Taxonomy

## Incidents
- Proxy/Upstream: 502/504, TLS, routing
- App exceptions: import errors, config, runtime
- Infra: systemd, permissions, disk, OOM
- Docker: build failure, restart loop, volume issues
- DB: connection, locks, migrations, constraints
- Security: suspicious requests, auth anomalies
- Performance: latency/CPU/mem/IO regressions

## Tasks
- Feature delivery
- Refactor/cleanup
- Observability/logging
- Deployment/migrations
